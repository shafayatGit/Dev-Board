Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML shows the texts with the HTML Tags.
2. innerText shows only the texts.
3. textContent shows all the things including the hidden text.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique where a single event listener is added to a parent element to handle events for multiple child elements using event bubbling. This improves performance and works for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a process in the DOM where an event starts from the target element and propagates up to its parent elements. This allows parent elements to handle events triggered by their children.
